# Job-Finder-web-App
JobFinder Near Me

JobFinder Near Me is a location-based job portal designed to help users discover and apply for jobs available in their local area. The application provides a clean and user-friendly interface for both job seekers and administrators.

Features:
User Registration & Login
Secure Authentication
Search Nearby Jobs
View Job Details
Apply for Jobs
Responsive User Interface
Admin Dashboard
Add, Edit, and Delete Job Listings
SQLite Database Integration
Flask Backend


Tech Stack:
Frontend: HTML5, CSS3, JavaScript
Backend: Python (Flask)
Database: SQLite
Version Control: Git & GitHub


Future Improvements:
Google Maps Integration
GPS-based Location Detection
Resume Upload
Email Notifications
Company Profiles
AI-based Job Recommendations
